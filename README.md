## Пакет game
 Реализована игра m,n,k с поддержкой **ошибок**, **двухкруговых турниров** (В рамках турнира каждый с каждым должен сыграть две партии), есть ромбическая доска для игры **Гекс** (с тремя направлениями линий). Также существуют **дополнительные ходы** (Если в результате хода игрока на доске появляется новая последовательность из 4+ одинаковых символов, то он делает дополнительный ход)

## Пакет markup
Набор классов для текстовой разметки. Дополнительно реализован метод `toTex`, генерирующий TeX-разметку:

    Абзацы предваряются командой `\par{}`
    Выделенный текст заключается в `\emph{` и `}`
    Сильно выделенный текст заключается в `\textbf{` и `}`
    Зачеркнутый текст заключается в `\textst{` и `}`

Добавлена поддержка:

      Нумерованных списков (класс `OrderedList`, окружение `enumerate`): последовательность элементов
      Ненумерованных списков (класс `UnorderedList`, окружение `itemize`): последовательность элементов
       Элементов списка (класс `ListItem`, тег `\item`: последовательность абзацев и списков



## Wspp.java, IntList.java, WsppLast.java
Это класс, подсчитывающий статистику встречаемости слов во входном файле. А также  использующий класс IntList, компактно хранящий список целых чисел.

#### Модификация (**WsppLast**) 

В выходном файле слова упорядочены по возрастанию длины, а при равной длине – по порядку первого вхождения во входной файл Вместо всех вхождений в файле надо указываются только последнее вхождение в строке. В словах могут дополнительно встречаться цифры и символы `$` и `_`

## ReverseMinC.java
Модификация к Reverse.java использующая мой сканер

Рассмотрим входные данные как (не полностью определенную) матрицу, вместо каждого числа выводится минимум из чисел, находящихся в его столбце в последующих строках, и его самого. Во вводе могут быть десятичные и восьмиричные числа  Числа дополнительно могут разделяться открывающими и закрывающими скобками


## WordStat.java
Это класс, подсчитывающий статистику встречаемости слов во входном файле.Словом называется непрерывная последовательность букв, апострофов (') и дефисов. Для подсчета статистики слова приводятся к нижнему регистру.
 Пусть _n_ – число слов во входном файле, тогда программа работает за O(_n_ log _n_).

 #### Модификация WordStatLengthMiddle.java
Назовём _серединой слова_ подстроку, полученную удалением первых и последних 3 символов слова. Слова длины меньшей 7 игнорируются.
 Выходной файл содержит все различные середины слов, встречающихся во входном файле, упорядоченные по возрастанию длины (при равенстве – по первому вхождению).



## Reverse.java
Это класс, читающий числа из стандартного ввода и выводящий их на стандартный вывод в обратном порядке. Порядок строк в выходе  обратен по сравнению с порядком строк во входе. Порядок чисел в каждой строке также обратен к порядку чисел во входе. Моя программа использует не более 4_M_ + 1024 байт памяти. (_M_ – объём памяти, необходимый для сохранения ввода в двумерном массиве `int` минимального размера)

#### Mодификация ReverseMaxC.java
Теперь смотрим на входные данные как (не полностью определенную) матрицу, вместо каждого числа выведите максимум из чисел, находящихся в его столбце в последующих строках, и его самого
